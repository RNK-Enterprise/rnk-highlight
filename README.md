# RNK™ Highlight

RNK™ Highlight adds hover glow effects to clickable scene objects in Foundry Virtual Tabletop.

## Installation

Use this manifest URL in Foundry VTT:

```text
https://github.com/RNK-Enterprise/rnk-highlight/releases/latest/download/module.json
```

1. Open **Add-on Modules** in Foundry VTT.
2. Select **Install Module**.
3. Paste the manifest URL.
4. Install the module and enable it in your world.

## What the module does

RNK™ Highlight adds configurable hover glow effects to supported scene objects so important interactable elements are easier to spot during play.

Supported object types:

- Tokens
- Tiles
- Drawings
- Walls
- Ambient lights

## Settings

All configuration is available to Game Masters from the scene controls.

- **Enable Glow**: Turns all hover effects on or off.
- **Glow Color**: Sets the glow color.
- **Intensity**: Adjusts glow strength from `0` to `100`.
- **Size**: Adjusts the glow radius from `0` to `50`.
- **Object Types**: Enables or disables glow per supported object type.

Setting changes apply immediately to hovered placeables.

## Troubleshooting

### Glow does not appear

1. Confirm **Enable Glow** is turned on.
2. Confirm the object type is enabled.
3. Try a brighter glow color.
4. Reload the world.

### Glow looks too strong

1. Lower **Intensity**.
2. Lower **Size**.
3. Disable glow for object types you do not need.
