# RNK Highlight

Adds hover glow effects to clickable scene objects in Foundry VTT.

## Installation

1. Download the module zip from releases.
2. Extract to `Data/modules/rnk-highlight`.
3. Enable in Foundry setup.

## Usage

- Hover over tokens, tiles, drawings, walls, or lights to see glow effects.
- GM can access settings via the lighting controls (lightbulb icon) to customize color, intensity, size, animation, and enabled object types.

## Settings

- **Enable Glow**: Toggle glow effects on/off.
- **Glow Color**: Choose glow color.
- **Intensity**: Adjust glow opacity (0-100).
- **Size**: Adjust glow blur size (0-50).
- **Animation**: Enable fade transitions.
- **Object Types**: Enable/disable glow for specific object types.

## Compatibility

Foundry VTT v11+.
